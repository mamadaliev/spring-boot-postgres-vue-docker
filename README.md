# spring-boot-postgres-vue-docker-example

## Stack
- Spring Boot 2.2.0
- Spring Data 2.2.0
- Postgres 9.4
- Docker 3
- Maven 4
- Tomcat 9

## Install
To build services and launch them in containers, you need to run `start.sh`.
```bash
$ bash start.sh
```
